About This App

This web app tracks the ratio between AXS/USDT and EOS/USDT cryptocurrencies from Binance every 5 minutes. It provides real-time updates and displays the current ratio.

Developed by Hocine Dev, this app uses Node.js for the backend to fetch data from the Binance API and JavaScript with HTML/CSS for the frontend.
